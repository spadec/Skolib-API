Описание сервисов API электронной библиотеки elib.skolib.kz

Все запросы отправляются методом POST с параметрами в теле запроса

https://dev.skolib.kz/elib/GetCats.php - Список категорий книг электронной библиотеки

Параметры:
status * - string Любое значение

/*************/

https://dev.skolib.kz/elib/GetBooksByCat.php - Список книг по ID-категории

Параметры:
catId * - integer ID-категории
orderby - string (id, title) - сортировка результатов по id и по title
direction - string (DESC, ASC) - направление сортировки
limit - integer кол-во возращаемых книг
offset - integer смещение относительно первого результата запроса

/*************/

https://dev.skolib.kz/elib/GetBookByID.php - Описание книги по ID-книги

Параметры:
bookId * - integer ID-книги

/*************/

https://dev.skolib.kz/elib/searchBooks.php - Поиск книг по электронной библиотеке

Параметры:
mode * - string (basic, advanced) - Режим поиска (Базовый или расширенный)
queryString - string строка поискового запроса только для базового поиска
author - string поиск по автору в расширенном поиске
title - string поиск по заглавию в расширенном поиске
pubyear - string год издания в строковом виде в расширенном поиске
limit - integer кол-во возращаемых статей
offset - integer смещение относительно первого результата запроса

/*************/

https://dev.skolib.kz/elib/setRaiting.php - Установить рейтинг книги

Параметры:
bookId * - integer ID-книги
raiting * - integer числовое значение рейтинга от 1 до 5

/*************/

https://dev.skolib.kz/elib/GetArticlesCats.php - Возвращает иерархический список категорий для статей

Параметры:
status * - string Любое значение

/*************/

https://dev.skolib.kz/elib/GetArticleCategoryByID.php - Возвращает категорию статьи по ID(либо категории либо статьи)

Параметры:
catId - integer ID категории
articleId - integer ID - статьи

/*************/

https://dev.skolib.kz/elib/GetArticlesByCat.php - Возвращает список статей категории

Параметры:
catId * - integer - ID категории статей
limit - integer кол-во возращаемых статей
offset - integer смещение относительно первого результата запроса

/*************/

https://dev.skolib.kz/elib/GetArticleByID.php - Возвращает полный текст статьи по ее ID

articleId * - integer - ID категории статей

/*************/

https://dev.skolib.kz/elib/GetRareBooks.php - Возвращает список "Книжные памятники"

Параметры:
limit - integer кол-во возращаемых статей
offset - integer смещение относительно первого результата запроса

/*************/

https://dev.skolib.kz/elib/searchArticles.php - Поиск статей по электронной библиотеке

Параметры:
queryString* - string строка поискового запроса для поиска статей
limit - integer кол-во возращаемых статей
offset - integer смещение относительно первого результата запроса

/*************/

https://dev.skolib.kz/elib/GetArticlesSubCats.php - Возвращает список вложенных категорий, если они есть

Параметры:
catId * - integer - ID категории статей
